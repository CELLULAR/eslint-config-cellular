ESLint config used at [CELLULAR](https://www.cellular.de)

## Usage

Install the conventions by running:

```
npm install --save-dev eslint eslint-config-cellular
```

Then add the extends to your `.eslintrc`:

```javascript
{
  "extends": "cellular"
}
```
